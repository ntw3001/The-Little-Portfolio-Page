What is BBQuery?

BBQuery is a small project built to assist players in the tabletop game Blood Bowl. It serves as a quick reference tool, allowing players to quickly look up dice tables during gameplay and check available options for match preparation.


It was written as a personal project using Ruby, built on a framework supplied by Le Wagon during their web development bootcamp.
